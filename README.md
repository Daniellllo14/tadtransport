# TadTransport — Webbplats

Detta projekt är en scaffold för `TadTransport` byggd med Vite + React + TypeScript och Tailwind CSS.

Snabbstart (Windows PowerShell):

```powershell
npm install
npm run dev
```

Bygg för produktion:

```powershell
npm run build
npm run preview
```

Notera:
- Lägg till din logotyp i `public/assets/logo.png` eller uppdatera `src/components` så den pekar rätt.
- Konfigurera Netlify/Vercel enligt instruktioner i `netlify.toml` eller i plattformens UI.
